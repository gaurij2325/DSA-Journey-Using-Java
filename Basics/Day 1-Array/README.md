# Arrays

This folder contains practice problems and revision of basic array concepts in Java.

## Concepts Covered
- Array Traversal
- Reverse Array
- Maximum & Minimum Element
- Frequency Count

---

## Files Included

- TraverseArray.java : Basic array traversal
- ReverseArray.java : Reverse array using brute force approach 
- MaxMinArray.java : Find maximum and minimum element in array 
- FrequencyCount.java : Frequency counting using brute force logic

---

## Key Learnings
- Understanding array indexing
- Importance of loop boundaries
- Debugging ArrayIndexOutOfBoundsException
- Difference between brute force and better approaches
